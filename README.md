# ABB_assignment_sample

## Start All 
In main folder execute
```
$ npm install
```
and 
```
$ npm start
```

if some permission error happen with script you have to execute
```
$ sudo chmod +x back.sh && sudo chmod +x front.sh 
```

## Start only backend
```
$ cd backend
$ npm install
$ nodemon & node robot.js
```

## Start only frontend
```
cd frontend
npm install
npm run start
```
